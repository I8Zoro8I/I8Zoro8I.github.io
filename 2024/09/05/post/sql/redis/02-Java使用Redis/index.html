Java使用Redis

安装

开始在 Java 中使用 Redis 前， 我们需要确保已经安装了 redis 服务及 Java redis 驱动，且你的机器上能正常使用 Java。 Java的安装配置可以参考我们的 Java 开发环境配置接下来让我们安装 Java redis 驱动：

  • 首先你需要下载驱动包 下载 jedis.jar，确保下载最新驱动包。
  • 在你的 classpath 中包含该驱动包。

本站提供了 2.9.0 jar 版本下载： jedis-2.9.0.jar
